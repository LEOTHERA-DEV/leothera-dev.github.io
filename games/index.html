AXOLOTL 40

AXOLOTL 40 is a first person game inspired by various arcade games.

The game is set in a distant future and in the same universe as Project: Axolus. AXOLOTL 40 mostly focuses on Earth as various industry giants put their new Mechs to the test in an interplanetary battle against alien races.

Project: Axolus

A series of retro styled action role-playing games featuring gameplay mechanics inspired by older RPGs and MMOs.
The first game, simply named Project: Axolus, is set in an alternate solar system and focuses on Saturn's lost race as they begin a long term mission of retrieving lost relics across the galaxy.
A free demo will be available on itch.io soon.

Vihltara

An upcoming Role Playing Game that combines African Fantasy with unique sci-fi elements.

The game is set in a world going through a series of events caused by beings from an unknown realm.
