Upcoming Games

AXOLOTL 40

AXOLOTL 40 is a first person game inspired by various arcade games.

Designed as an indirect sequel to Project: Axolus. AXOLOTL 40 focuses on Earth's industrial giants as they put their new Mechs to the test in an interplanetary battle against alien races.

Project: Axolus

A series of retro styled action role-playing games featuring gameplay mechanics inspired by older RPGs and MMOs.
The first game, simply named Project: Axolus, is set in an alternate solar system in which Saturn's remaining inhabitants begin a long term mission of retrieving lost relics across the galaxy.
A free demo will be available on itch.io soon.

Project: Edith

Edith (Working Title) is an first person narrative driven game with a darker tone and plot. The game aims to explore various conditions such as anxiety, depression and gender dysphoria by being set in a world where the protagonist has to live a life in a world that never remains constant and is difficult to keep up with.
Unlike the other games I'm working on, this one combines visual novel elements with an interactable world, giving you complete freedom to explore various areas without being constrained by the main story.

Edith currently does not have a release date.

Vihltara

An upcoming Role Playing Game that combines African Fantasy with unique sci-fi elements.

The game is set in a world going through a series of events caused by beings from an unknown realm.
